# Simple Interest Calculator

Sebuah aplikasi sederhana berbasis script untuk menghitung bunga tunggal (Simple Interest) berdasarkan modal awal, suku bunga tahunan, dan jangka waktu.

## Deskripsi
Proyek ini dibuat untuk memenuhi tugas praktikum pemrograman. Algoritma ini menghitung total bunga yang dihasilkan selama periode tertentu tanpa memperhitungkan bunga majemuk.

## Rumus Matematika
Perhitungan ini menggunakan rumus standar bunga tunggal:

$$I = P \times r \times t$$

**Keterangan:**
* **I**: Jumlah bunga (Interest)
* **P**: Modal awal (Principal amount)
* **r**: Suku bunga tahunan (Annual interest rate dalam desimal)
* **t**: Jangka waktu dalam tahun (Time period in years)
